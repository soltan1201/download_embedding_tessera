Artigos científicos:

    "Representation Learning with Contrastive Predictive Coding" (https://arxiv.org/abs/1905.09276): Este artigo apresenta o Contrastive Predictive Coding (CPC), uma técnica de aprendizado auto supervisionado que pode ser usada para treinar autoencoders.
    "Deep Convolutional Autoencoders for Image Denoising" (https://www.cs.toronto.edu/~vinod/papers/autoencoder.pdf): Este artigo explora o uso de autoencoders convolucionais para remover ruído de imagens, demonstrando sua capacidade de aprender representações robustas.
    "Variational Autoencoders" (https://arxiv.org/abs/1312.6114): Este artigo introduz os Autoencoders Variacionais (VAEs), uma abordagem que permite gerar novas amostras a partir do espaço de embeddings.


    Projetos no GitHub:

    "Autoencoder in PyTorch" (https://github.com/L1aoXingyu/Autoencoder-in-PyTorch): Este repositório contém implementações de diferentes tipos de autoencoders (básico, convolucional, variacional) usando o framework PyTorch.
    "TensorFlow Autoencoder" (https://github.com/jtoy/tensorflow-autoencoder): Este repositório oferece exemplos de como construir e treinar autoencoders com TensorFlow, incluindo exemplos para diferentes tarefas.
    "Keras Autoencoder Examples" (https://github.com/keras-team/keras/tree/master/examples/conv_autoencoder): Este repositório oficial do Keras contém exemplos de autoencoders convolucionais para diferentes aplicações, como remoção de ruído e compressão de imagens.
